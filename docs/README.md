# 기능 목록 정리

1. 게임 시작 전
    - 게임 시작 문구 출력
2. 게임 시작
    - 상대방 숫자 랜덤 저장
    - 사용자 숫자 제시
    - 경우에 따른 문구 출력
        - 자리와 숫자 모두 같으면 스트라이크
        - 자리는 다르고 숫자가 같으면 볼
        - 아무것도 같지 않으면 낫싱
        - 3개 숫자 모두 정답일 경우 게임 종료
3. 게임 종료
    - 게임 종료 후 또 다른 게임 진행 또는 프로그램 종료 선택
4. 입력 예외 사항
    - 숫자가 아닐 경우
    - 길이가 3이 아닌 경우
    - 중복 숫자를 입력할 경우
    - 게임 종료후 1,2를 선택하지 않을 경우
