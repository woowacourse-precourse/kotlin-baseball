## 🚀 기능 요구 사항

##  게임 시작

- 게임 시작
  - 컴퓨터에 임의의 수 3개 부여
  - 게임 실행
  - 게임 재시작
- 임의의 수 3개 생성하는 함수
  - camp.nextstep.edu.missionutils.Randoms 을 사용해 각각 다른 숫자 3 개를 부여함
- 게임
  - Console로 입력을 받는다.
  - 입력받은 값의 예외 처리를 한다
  - 결과값을 계산한다.
  - 결과값을 출력한다
  - 결과값이 게임종료 조건이면 이 함수를 끝낸다.
- 예외 처리
  - 입력값이 3자리가 아닐 때
  - 입력값이 중복되었을 때
  - 입력값이 숫자가 아닐 때
