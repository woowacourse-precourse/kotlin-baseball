## 🎯기능 목록

1. 게임 시작 멘트 출력
2. 컴퓨터가 정답숫자를 랜덤으로 생성
3. 사용자가 숫자를 입력
4. 사용자가 입력한 숫자를 비교
   1. 사용된 숫자는 맞는데 자리가 틀릴 때
   2. 사용된 숫자도 자리도 맞추었을 때
   3. 사용된 숫자를 하나도 맞추지 못했을 때
5. 사용자가 정답숫자와 일치하게 입력한다면 축하 멘트 및 게임 재시작 의사 확인
6.  사용자가 잘못된 값을 입력할 경우 `IllegalArgumentException` 발생시킨 후 애플리케이션 종료
